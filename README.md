# Solutions for high school diploma exams

Here you can find different problems and solutions. On the subjects that are in the high school diploma exams.

The solution in the conditions.py is for when you are given a sentence that says where a number shoud be and the programs outputs the condition. For example if you want to know what are the conditions for a number that is in (2,5] the program will output the condition in C: (x>2) && (x<=5).
